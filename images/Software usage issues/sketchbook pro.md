# 卸载sketchbook后，无法再次安装
![电脑安装软件失败但提示已安装](./images/1642932545591.png)
解决办法：
1. 找到SketchBookPro_2021_ML_Win_64bit_dlm.sfx解压的路径。
2. 运行SketchBookPro_2021_ML_Win_64bit_dlm\x64\SBP下的SBP.msi，然后选择卸载（remove）即可。
3. 使用垃圾清理软件，删除注册表。
