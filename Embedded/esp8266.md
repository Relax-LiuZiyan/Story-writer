# ATK-ESP8266 配置

注意：烧录固件时ESP8266供电需要为5V，并且需要管理员身份运行烧录软件进行烧录固件。

IO_0烧录的时候连接低电平，烧录软件提示同步时候，需要RST引脚瞬间接地。
``` c?linenums
AT+RESTORE
AT+CWMODE=1
AT+RST
AT+CWJAP="VIP-WIFI","123512liu"
AT+ATKCLDSTA="33596558792964906177","12345678"

AT+RESTORE
```
